## 四、生产部署（Linux 服务器）

### 4.1 安装依赖

```bash
# 系统依赖
sudo apt update && sudo apt install python3.10 python3.10-venv nginx -y

# 项目依赖
cd /opt/tikpan_web
python3 -m venv venv
source venv/bin/activate
pip install gunicorn flask flask-cors requests pillow python-dotenv
```

### 4.2 Gunicorn 服务

```bash
# 创建 systemd 服务
sudo tee /etc/systemd/system/tikpan-web.service <<EOF
[Unit]
Description=Tikpan AI Studio Web
After=network.target

[Service]
User=www-data
WorkingDirectory=/opt/tikpan_web
Environment="PATH=/opt/tikpan_web/venv/bin"
EnvironmentFile=/opt/tikpan_web/.env
ExecStart=/opt/tikpan_web/venv/bin/gunicorn \
    --workers 4 \
    --bind 127.0.0.1:5000 \
    --timeout 300 \
    --access-logfile /var/log/tikpan_web/access.log \
    --error-logfile /var/log/tikpan_web/error.log \
    app:app

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl enable tikpan-web
sudo systemctl start tikpan-web
```

### 4.3 Nginx 反向代理

```nginx
# /etc/nginx/sites-available/tikpan_web

server {
    listen 80;
    server_name 你的域名.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name 你的域名.com;

    ssl_certificate /etc/letsencrypt/live/你的域名.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/你的域名.com/privkey.pem;

    client_max_body_size 200M;   # 支持上传参考图

    # 静态文件缓存
    location /static/ {
        root /opt/tikpan_web;
        expires 30d;
        add_header Cache-Control "public, no-transform";
    }

    # 本地存储的生成文件
    location /outputs/ {
        root /opt/tikpan_web/data;
        add_header Cache-Control "public, max-age=86400";
    }

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_read_timeout 300s;
        proxy_connect_timeout 10s;
    }
}
```

## 六、扩展接口规范（方便后续新增模型）

新增任何模型，只需 3 步：

**Step 1：在 `models/__init__.py` 的 `seed_pricing()` 里加入定价行**

```python
# 图片模型示例
("new-image-model", "新模型名称", credits_1k, credits_2k, credits_4k)

# TTS 模型示例（per_char）
("new-tts-model", "新TTS名称", per_1k_chars, min_credits)

# 视频/音乐模型示例（per_call）
("new-video-model", "新视频名称", credits_per_call)
```

**Step 2：在 `backend/handlers.py` 里添加 payload 构建函数**

```python
def build_new_model_payload(model_id, data):
    return {
        "model": model_id,
        "prompt": data.get("prompt", ""),
        # ... 该模型的其他参数
    }
```

**Step 3：在对应 API 端点里增加路由**

```python
# api/generate.py（图片）或 api/audio.py（音频）或 api/video.py（视频）
elif model_id == "new-model":
    payload = build_new_model_payload(model_id, data)
    endpoint = "/v1/新端点"
```

就这样，新模型自动获得：计费/余额扣减/幂等保护/日志记录/OSS存储。
