# Tikpan Web

Tikpan Web is the React + Vite frontend and demo platform API for the Tikpan AI aggregation workbench.

## Frontend

```bash
npm ci
npm run dev
npm run build
```

## API

```bash
cd api
npm ci
npm run dev
```

The API package also includes a worker and validation scripts:

```bash
npm run worker
npm run check
```

See `ARCHITECTURE.md`, `INTEGRATION.md`, and `api/DEPLOYMENT.md` for platform design, integration notes, and deployment details.
